## 🔁 Critical reminders specific to ADR-369

1. **Units**: Floor uses METRES, BIM entities use MILLIMETRES. Conversion layer ΑΠΑΡΑΙΤΗΤΟΣ στα factory functions.
2. **Floor.elevation semantic CHANGES**: Πριν = absolute z. Τώρα (Q2) = RELATIVE to building.baseElevation. World z = `survey.z + project.basePoint.z + building.baseElevation + floor.elevation`.
3. **Wall.topBinding='storey-ceiling'** = `floor.elevation + floor.height`. Need to compute correctly.
4. **IFC GUID stability**: Generate ONCE on entity create, NEVER regenerate. Stored in Firestore.
5. **Greek ordinal generator**: Reuse existing `useFloorNumberToText.ts` logic (1→1ος, 11→11ος, 21→21ος). Do NOT duplicate.
