ΕΝΤΟΛΗ ΠΡΟΣ ΠΡΑΚΤΟΡΑ (CLAUDE / ANTHROPIC)

ΑΠΑΓΟΡΕΥΕΤΑΙ ΡΗΤΑ να:

σκοτώνεις διεργασίες (kill / taskkill / pkill / stop-server),

αλλάζεις πόρτα,

ανοίγεις νέα ports,

επανεκκινείς dev server,

τρέχεις npm run dev, npm start, yarn dev, pnpm dev ή αντίστοιχες εντολές.

ΥΠΟΧΡΕΩΣΗ:

Χρησιμοποιείς ΑΠΟΚΛΕΙΣΤΙΚΑ την ήδη ενεργή πόρτα στην οποία τρέχει η εφαρμογή.

Θεωρείς ότι ο dev server τρέχει σωστά εκτός αν σου ζητηθεί ρητά το αντίθετο.

Κάνεις ΜΟΝΟ στατικές αλλαγές κώδικα (αρχεία .ts, .tsx, .css, κ.λπ.).

Δεν πειράζεις διεργασίες, ports, terminals ή background tasks.

ΑΝ χρειαστεί αλλαγή που κανονικά απαιτεί restart:

ΣΤΑΜΑΤΑΣ.

Το αναφέρεις ρητά.

Δεν εκτελείς καμία εντολή.
