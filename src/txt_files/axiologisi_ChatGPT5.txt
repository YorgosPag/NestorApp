Όχι με τα δεδομένα που έχω. Είναι “Enterprise” μόνο αν ισχύουν ΟΛΑ τα κάτωθι· αν λείπει έστω ένα, δεν είναι ολοκληρωμένο.

Jest (μονάδες)

jsdom περιβάλλον (header ή config).

Ονομαστικοί listeners + σωστό removeEventListener.

Σταθερό event schema: { ids: string[], mode: 'replace|add|remove' }.

Deterministic timers: jest.useFakeTimers() + jest.advanceTimersByTime().

DOM-ορατές assertions (π.χ. μετρητής grips).

jest-axe χωρίς παραβιάσεις.

Perf test σε pure selection.

Playwright (E2E “ό,τι βλέπει ο χρήστης”)

Το spec του canvas υπάρχει και περνά.

Visual snapshots committed και σταθεροποιημένα.

Traces/videos on-failure ενεργά.

axe scan σε βασικά flows.

Multi-browser projects (Chromium/Firefox/WebKit).

CI workflow με retries και artifacts.

CI πολιτικές

Coverage threshold ορισμένο.

Flaky quarantine ή τουλάχιστον retries per-test.

PR gate που μπλοκάρει αν αποτύχει Jest ή Playwright.

Γρήγορος αυτοέλεγχος
npm test                    # όλα πράσινα και axe 0 violations
npx playwright test         # περνούν όλα τα browsers
ls playwright-report        # υπάρχει report
ls test-results             # υπάρχουν traces/videos
