## 0. TL;DR

Η εφαρμογή μας είναι **BIM authoring + preliminary tributary design** (Revit-authoring parity, με auto
οπλισμό/πέδιλα/sizing). Το επαγγελματικό τεύχος είναι **πλήρης 3D FEM solver + δυναμική φασματική σεισμική
ανάλυση EC8 + ικανοτικός σχεδιασμός + παραδοτέο τεύχος**. Η Revit κάνει το πρώτο· για το δεύτερο κάνει
export στο **Robot Structural Analysis**. **Το μεγάλο κενό μας = αναλυτικό μοντέλο + solver + σεισμός EC8.**

Η σωστή Revit-grade αρχιτεκτονική: χτίζουμε ένα **Analytical Model SSoT** (παράγωγο του υπάρχοντος
structural-organism graph, ADR-459) και πάνω του solver → σεισμό → ελέγχους → τεύχος. Roadmap σε §4.

## 1. ΤΙ ΕΙΝΑΙ ΤΟ ΤΕΥΧΟΣ (Statics 2025, Ευρωκώδικες)

**Κτίριο:** πολυκατοικία Ο.Σ., 5 στάθμες (z=-1.40 υπόγειο / 0.00 / 3.00 / 6.00 / 9.55 m), τοιχώματα +
περιμετρικά τοιχεία υπογείου. Υλικά C25/30, B500C.

### 1.1 Μεθοδολογία ανάλυσης (Τεχνική Έκθεση, σελ 3-7)
- **3D χωρικό πλαίσιο** (space frame FEM): μητρώα κόμβων / μελών / δυσκαμψίας → επίλυση.
- **Δυναμική φασματική ανάλυση EC8**: **279 ιδιομορφές**, μέθοδος επαλληλίας **CQC**, φάσμα σχεδιασμού EC8.
- **Σεισμός**: Ζώνη I (a=0.16g), γI=1.0, **ΚΠΜ** (μέση πλαστιμότητα), q=2.0 x/y, έδαφος B (S=1.2,
  Tb=0.15, Tc=0.50, Td=2.50), ιδιοπερίοδοι Tx=0.36/Ty=0.35, Sd=2.35. ψ1=0.5, ψ2=0.3.
- **Μάζες**: M=(G+ψ2·Q)/9.81· ανά στάθμη M (Mg), Jm (ροπή αδράνειας μάζας, Mg·m²), Is (ακτίνα αδράνειας).
- **Σεισμικοί συνδυασμοί Newmark**: X = ±1.0·Xx ±0.3·Xy και ±0.3·Xx ±1.0·Xy.
- **Συνδυασμοί**: στατικός `1.35G + 1.50Q`· σεισμικός `G + ψ2·Q ± E(G+φ·ψ2·Q)` (EN1990 A1 + EN1998-1 §4.2.4).
- **Πλάκες**: ροπές μέθοδος **Czerny** (πίνακες)· αντιδράσεις DIN 1045 (γεωμετρικός μερισμός επιφανειών
  φόρτισης → γραμμικά φορτία περιμετρικών δοκών)· ροπές ανοίγματος EC2.
- **Θεμελιώσεις**: ικανοτικά EC8 §4.4.2.6 (4.30) `Efd = Efg + γRd·Ω·Efe`· πεδιλοδοκοί/κοιτόστρωση με
  μοντέλο **Winkler** (ελατήρια εδάφους, δείκτης K)· έλεγχος αδρανούς επιφάνειας <50%· ανηγμένες
  εκκεντρότητες `εx²+εy² < 1/9` (1/16 σεισμ. ευπαθή).
- **Περιμετρικά τοιχεία υπογείου**: προσομοίωση χιαστί άκαμπτων ράβδων· ώθηση γαιών **Coulomb** (ηρεμία)
  + **Mononobe-Okabe** (σεισμός)· διαστασιολόγηση ως τετραέρειστες πλάκες Czerny.

### 1.2 Διαστασιολόγηση (EC2 — «συνολική αντοχή»)
Κάμψη+αξονική, διάτμηση+**στρέψη** (Tsd), διάτρηση πεδίλων, **λυγισμός** κατακόρυφων (λmax=10.78/√νd),
ΟΚΛ ρηγματώσεις+βέλη. Output δοκού (δείγμα): `M_Rd / M_Sd / As,req / As,tot (cm²) / ρ,ρ',ρmin,ρmax /
Tsd / [ΑΚΡΟ Α,Β: Vsd, Vrd,s, VrdMax] / κ4Φ20 π4Φ20 (κάτω/άνω) / συνδετήρες Φ10/15 4τμητοι`.

### 1.3 Γενικοί έλεγχοι EC8 (§4 Τεχνικής Έκθεσης)
i) αποφυγή **μηχανισμού ορόφου** (soft-storey) · ii) επάρκεια & καλή τοποθέτηση **τοιχωμάτων** ·
iii) **επιρροές 2ας τάξης** `Θ = Ptot·dr/(Vtot·h) < 0.10` · iv) αποφυγή **ψαθυρής διατμητικής αστοχίας**
(ικανοτική τέμνουσα) · v) **ευστρεψία** ορόφων (r ≥ ls) · vi) **περίσφιξη** υποστυλωμάτων (ωwd) ·
vii) **κοντό υποστύλωμα** (short column).

### 1.4 Παραδοτέο τεύχος (TOC, 18 ενότητες, 261 σελ)
Τεχνική Έκθεση · Σχέδια Σταθμών · 3D · **Μητρώα** (Κόμβων/Μελών/Φορτίων/Εντατικών Μεγεθών) · Αντισεισμικά
Στοιχεία Σταθμών (μάζες/Jm/Is/δυσκαμψίες/κέντρα) · Δυναμική Ανάλυση · **Διαγράμματα Μ/V/N** δοκών+υποστυλ. ·
**Διαστασιολόγηση** Πλακών/Δοκών/Υποστυλωμάτων ανά στάθμη · Έλεγχος Επάρκειας Τοιχωμάτων · Έλεγχος Θ ·
Έλεγχοι Κόμβων (ικανοτικός) · **Προμέτρηση Υλικών** (m³ σκυρόδεμα, kg χάλυβας — 88.6 kg/m³).

## 3. GAP ANALYSIS — ΤΙ ΜΑΣ ΛΕΙΠΕΙ (κατηγοριοποιημένο)

### A. Ανάλυση & Σεισμικά (το ΜΕΓΑΛΟ κενό) — 0% σήμερα
1. **3D FEM space-frame solver** (μητρώο δυσκαμψίας, K·u=F) → πραγματικά M/V/N.
2. **Μάζες ορόφων** M=(G+ψ2Q)/g, **διάφραγμα** (rigid diaphragm), κέντρο μάζας/δυσκαμψίας, στατική +
   **τυχηματική εκκεντρότητα**.
3. **Δυναμική φασματική EC8**: eigenvalue (ιδιομορφές/ιδιοπερίοδοι), φάσμα σχεδιασμού EC8, **CQC**,
   ποσοστά συμμετοχής δρώσας μάζας (≥90%).
4. **Σεισμικά φορτία/συνδυασμοί** (Newmark ±1.0/±0.3, σεισμικός συνδυασμός με φ·ψ2).
5. **Διαγράμματα M/V/N** (envelope ανά μέλος/συνδυασμό).

### B. Έλεγχοι EC8 (capacity design) — 0% σήμερα
6. **Έλεγχος θ** (P-Δ, επιρροές 2ας τάξης): Θ=Ptot·dr/(Vtot·h)<0.10.
7. **Σχετικές μετακινήσεις ορόφων** (interstorey drift) + **έλεγχος βλαβών** (περιορισμός dr).
8. **Ευστρεψία** ορόφων (r ≥ ls).
9. **Soft-storey** (αποφυγή μηχανισμού ορόφου).
10. **Ικανοτική τέμνουσα** δοκών/υποστυλωμάτων (ψαθυρή αστοχία).
11. **Έλεγχος κόμβων** (ΣM_Rc ≥ 1.3·ΣM_Rb — ισχυρό υποστύλωμα/ασθενής δοκός).
12. **Περίσφιξη** υποστυλωμάτων (ωwd, μηχανικό ποσοστό), **κοντό υποστύλωμα**, **λυγισμός** (λmax).

### C. Δομικά στοιχεία/φορτία που λείπουν
13. **Τοιχώματα (shear walls)** ως αναλυτικά στοιχεία + **έλεγχος επάρκειας EC8**.
14. **Περιμετρικά τοιχεία υπογείου**: ώθηση γαιών Coulomb + Mononobe-Okabe, διαστασιολόγηση τετραέρειστης.
15. **Πεδιλοδοκοί/γενική κοιτόστρωση επί Winkler** (ελαστικό έδαφος, ελατήρια K) — ADR-464 έχει strip/raft
    αλλά ΟΧΙ Winkler grid solve.
16. **Ικανοτικά θεμέλια** (Efd=Efg+γRd·Ω·Efe) + εκκεντρότητες (εx²+εy²<1/9, αδρανής επιφ.<50%).
17. **Στρέψη δοκών (Tsd)** στον οπλισμό.
18. **Γραμμικά φορτία τοιχοποιίας** σε δοκούς (μπατικές 3.6 / δρομικές 2.1 kN/m² → kN/m στη φέρουσα δοκό).
    Σήμερα τα φορτία μας είναι μόνο επιφανειακά (occupancy).
19. **Φορτία χιονιού (EN1991-1-3)** + **ανέμου (EN1991-1-4)**.
20. **Σκάλες** ως φέρον στοιχείο + φορτίο (load κλιμακοστασίων 3.5 kN/m²).

### D. Παραδοτέα
21. **Τεύχος υπολογισμών** (PDF εγκρίσιμο): τεχνική έκθεση + μητρώα + διαγράμματα + διαστασιολόγηση +
    έλεγχοι + προμέτρηση.
22. **Προμέτρηση υλικών** συνολική (σκυρόδεμα m³ ανά στοιχείο/στάθμη, χάλυβας kg, kg/m³, ξυλότυποι m²).

## 4. ROADMAP (Revit-grade + Full SSoT) — προτεινόμενες φάσεις

> **Αρχή σχεδίασης (Revit):** όλα κρέμονται από ένα **Analytical Model** ξεχωριστό από το physical BIM,
> παραγόμενο/συγχρονισμένο με τον υπάρχοντα `structural-organism` graph (ADR-459). Κάθε νέο feature =
> SSoT module + proactive σύνδεση στην αλυσίδα, όπως κάναμε σε loads/reinforce/sizing.

| Tier | Περιεχόμενο | Εξάρτηση | Μέγεθος |
|---|---|---|---|
| **T1 — Authoring/Loads completeness** | wall line-loads σε δοκούς (#18), στρέψη Tsd (#17), σκάλες (#20), χιόνι/άνεμος (#19) | καμία (πάνω στο υπάρχον takedown) | 🟢 Μεσαίο, υψηλό value/χαμηλό ρίσκο |
| **T2 — Analytical Model SSoT** | nodes/members(stiffness)/supports/diaphragm/load-cases/EC0+EC8 combinations από τον organism graph | T1 | 🟡 Μεγάλο — **το θεμέλιο** |
| **T3 — Static FEM solver** | 3D K·u=F, πραγματικά M/V/N, diagrams (#1,#5) | T2 | 🔴 Μεγάλο |
| **T4 — Seismic EC8 (modal RS)** | μάζες/διάφραγμα/eigenvalue/φάσμα/CQC/Newmark (#2,#3,#4) | T3 | 🔴 Μεγάλο |
| **T5 — EC8 checks & capacity design** | θ, drift, ευστρεψία, soft-storey, ικανοτική τέμνουσα, κόμβοι, περίσφιξη, κοντό υποστύλωμα (#6-#12) | T4 | 🔴 Μεγάλο |
| **T6 — Foundations/walls advanced** | Winkler grid, ικανοτικά θεμέλια, εκκεντρότητες, shear-wall EC8, τοιχεία υπογείου Coulomb/M-O (#13-#16) | T3/T4 | 🔴 Μεγάλο |
| **T7 — Παραδοτέο τεύχος** | PDF report + μητρώα + διαγράμματα + προμέτρηση (#21,#22) | T3+ | 🟡 Μεσαίο (reuse detail-sheet SSoT) |

**Ρεαλιστική σύσταση προτεραιότητας:**
1. **T1** πρώτα — άμεση βελτίωση ακρίβειας των φορτίων που ΗΔΗ έχουμε (wall line-loads = το πιο εμφανές
   κενό· οι τοιχοποιίες είναι κύριο μόνιμο φορτίο δοκών). Χαμηλό ρίσκο, μεγάλη απόδοση, καμία solver εξάρτηση.
2. **T2** — Analytical Model SSoT: το στρατηγικό θεμέλιο. Χωρίς αυτό, T3-T6 δεν γίνονται.
3. **T7** μπορεί να ξεκινήσει νωρίς (παραδοτέο τεύχος των ΥΠΑΡΧΟΝΤΩΝ tributary αποτελεσμάτων — δίνει
   αξία ακόμη πριν τον solver).
4. **T3→T4→T5** = η μεγάλη επένδυση (πραγματικός FEM + σεισμός). Εδώ αξίζει να εξεταστεί και **export σε
   open solver** (π.χ. OpenSees/ASCII) ως εναλλακτική του ίδιου solver — όπως Revit→Robot.
